- 여러 약속들을 관리하기 어려우셨나요? 여러 약속들을 하나의 앱으로 관리해 보세요.
- 지각하지 않으려면 언제 나가야할지 모르겠나요? 오디가 약속에 늦지 않도록 출발할 시간에 알림을 보내드려요.
- 친구가 언제 도착할지 궁금하신가요? 친구의 실시간 위치 정보를 기반으로 친구가 언제 도착할지 알려드려요.
- 지각 예정이거나 지각인 친구를 재촉해 보세요.
